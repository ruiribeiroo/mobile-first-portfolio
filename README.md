# Portfólio Pessoal

Site portfólio pessoal desenvolvido para apresentar informações sobre o programador, stack tecnológica utilizada e projectos desenvolvidos. O site possui design moderno e responsivo, com suporte a modo escuro/claro e navegação intuitiva entre as diferentes secções.

## 📋 Descrição

Este projecto é um portfólio pessoal desenvolvido em HTML, CSS e JavaScript vanilla. O site apresenta informações profissionais, experiência, formação, stack tecnológica e projectos desenvolvidos. Inclui funcionalidades como modo escuro/claro com persistência de preferência no navegador, design responsivo e navegação fluida entre as páginas.

## 🚀 Stack Tecnológica

### Frontend

- **HTML5** - Estrutura semântica do site
- **CSS3** - Estilização com variáveis CSS e design responsivo
- **JavaScript (Vanilla)** - Funcionalidades interactivas e gestão de tema

### Ferramentas e Bibliotecas

- **Git** - Controlo de versão
- **VS Code** - Editor de código
- **GitHub** - Plataforma de hospedagem de código

### Fontes

- **Poppins** - Fonte Google Fonts utilizada no projecto

## 📁 Estrutura do Projecto

```
projeto-portfolio/
├── assets/
│   ├── css/

│   │   └── main.css          # Estilos principais
```

## ✨ Funcionalidades

- ✅ **Modo Escuro/Claro** - Alternância entre temas com persistência no localStorage
- ✅ **Design Responsivo** - Adaptação para diferentes tamanhos de ecrã (Apartir da série Mobile First)
- ✅ **Navegação Intuitiva** - Menu de navegação com links para todas as secções
- ✅ **Secções Informativas**:
  - Home com apresentação pessoal
  - Sobre mim com experiência e formação
  - Stack tecnológica
  - Projectos desenvolvidos
  - Contacto
- ✅ **Links para Redes Sociais** - Integração com GitHub, LinkedIn e Twitter/X

## 🛠️ Como Utilizar

1. Abra o projecto no Visual Studio Code
2. Instale a extensão "Live Server" se ainda não tiver instalada
3. Clique com o botão direito no ficheiro `index.html` e seleccione "Open with Live Server"
4. O site será aberto automaticamente no seu navegador

## 🎨 Personalização

### Cores e Tema

As cores e variáveis de tema podem ser personalizadas no ficheiro `assets/css/_variables.css`.

### Conteúdo

- Edite os ficheiros HTML para actualizar o conteúdo
- Adicione as suas próprias imagens na pasta `assets/images/`
- Modifique os links de redes sociais nos ficheiros HTML
