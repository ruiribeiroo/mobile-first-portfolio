# Portfólio Mobile-First

Um website de portfólio pessoal desenvolvido com foco em design responsivo e otimização móvel. Este repositório contém os arquivos HTML, CSS e JavaScript necessários para exibir informação sobre projetos, contatos e habilidades.

## Descrição

O objetivo deste projeto é apresentar o trabalho e competências de Rui Ribeiro através de uma interface simples, leve e adaptada a todos os tamanhos de ecrã, especialmente dispositivos móveis. As páginas incluem:

- `index.html` – página inicial com introdução e navegação
- `about.html` – informação sobre mim e experiência
- `projects.html` – lista de projetos realizados
- `stack.html` – tecnologias e ferramentas utilizadas
- `contact.html` – formulário ou detalhes de contacto

O CSS está organizado na pasta `assets/css`, com  `_variables.css` para variáveis de estilo, e `main.css` para regras principais. O JavaScript está em `assets/js/main.js` para funcionalidades interativas.

## Funcionalidades

- Layout responsivo **mobile-first**
- Utilização de variáveis CSS para consistência de estilos
- Menu de navegação simples e claro
- Imagens otimizadas na pasta `assets/images`
- Formulário de contacto ou informações de e‑mail

## Começando (Instalação)

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/mobile-first-portfolio.git
   cd mobile-first-portfolio
   ```
2. Abra `index.html` no navegador de sua preferência. Não há dependências externas ou build necessário.

## Desenvolvimento

Este projeto não utiliza ferramentas de build complexas. Para editar:

- Modifique os arquivos HTML na raiz
- Ajuste estilos em `assets/css/main.css` ou `_variables.css`
- Adicione/edite scripts em `assets/js/main.js`

Teste visualmente abrindo os ficheiros no browser e usando o modo de responsividade das ferramentas de desenvolvedor.
